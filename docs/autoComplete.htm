CalvinUI

AutoComplete(自动补全文本框)

依赖
脚本文件:jquery.js
参数
  • options
    Object
    可选

    多选形式的可配置参数。

以下是option参数的说明

参数 数据类型 默认值 说明
min Number 1

智能提示的文本长度
source Array []

静态数据
dynamicSource Boolen false

采用ajax形式动态数据(如果是ajax获取数据一定要true)
ajaxOption Object :$.ajaxSettings

ajax请求参数配置
selected Function 空函数

2个参数(event,ui),第一个参数是点击事件,第二个参数是点击的项,具有text,value属性

实例1

基本用法 使用静态数据

<script type="text/javascript">
            $("#basic").CalvinAutoComplete({
                AutoInput: false,
                MenuHideAuto: false,
                min: 1,
                source: ["许志伟1", "许志伟2", "许志伟3", "许志伟4", "许志伟5"]
            });
 </script>

输入"许"看看 可上下键操作

实例2

基本用法 使用事件

<script type="text/javascript">
  $("#TxtEventTest").CalvinAutoComplete({ styleInfo: { width: 250 },
                min: 1,
                source: [
              { "text": "许志伟1", "value": "沙发上" },
              { "text": "许志伟2", "value": "3" },
              { "text": "许志伟3", "value": "3" }
              ],
                selected: function (event, ui) {
                    alert(ui.text);
                    alert(ui.value);
                }
            });
 </script>

输入"许"看看 可上下键操作

实例3

基本用法 获取远程数据

<script type="text/javascript">
 $("#txtAjax").CalvinAutoComplete({
                min: 1,
                dynamicSource: true,
                ajaxOption: { url: "autoCompleteJson.json", dataType: "json", data:
                {
                    name: "xuzhiwei",
                    age: 11
                }
                }
            });
 </script>

输入"许"看看 可上下键操作

  • AutoComplete(自动补全)
  • Accordion(手风琴菜单)
