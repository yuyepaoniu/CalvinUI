CalvinUI

Drag(拖拉)

依赖
脚本文件:CalvinDraggable.js
参数
  • options
    Object
    可选

    多选形式的可配置参数。

以下是option参数的说明

参数 数据类型 默认值 说明
containment HTMLDomElement null

拖拉的限定容器范围
axis string('v'|'h') null

拖拉的限定方向
handle string selector|HTMLDOMEelement

拖拉的手柄
proxy Boolean false

是否设置拖拽元素的代理模式，代理模式时并不会移动拖拽元素本身，仅会出现一个同尺寸的代理元素，如果拖拽元素本身的内容太多可能拖拽性能不好，此时可以使用代理模式。

实例1

基本用法

<script type="text/javascript">
$("#drag_basic").CalvinDraggable();
 </script>

实例2

限定容器

<script type="text/javascript">
  $("#drag_container").CalvinDraggable({ containment: document.getElementById("dragContainer") });
 </script>

实例3

使用代理元素

<script type="text/javascript">
  $("#drag_proxy").CalvinDraggable({ proxy:true });
 </script>
