CalvinUI

Combobox(组合框)

依赖
脚本文件:CalvinComboBox.js
样式文件:combobox.css
参数
  • options
    Object
    可选

    多选形式的可配置参数。

以下是option参数的说明

参数 数据类型 默认值 说明
min Number 1

智能提示的文本长度

source Array []

静态数据

selected Function 空函数

2个参数(event,ui),第一个参数是点击事件,第二个参数是点击的项,具有text,value属性

实例1

<script type="text/javascript">
   $(document).ready(function () {
            $("#Text1").CalvinComboBox({
                min: 1,
                source: [{ "text": "许志伟1", "value": "1" }, { "text": "许志伟2", "value": "2" }, { "text": "陈志伟2", "value": "2" }, { "text": "许志伟3", "value": "3" }]
            , selected: function (event, ui) {
                alert(ui.text);
                alert(ui.value);
            }
            });
        });
 </script>
