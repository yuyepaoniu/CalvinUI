CalvinUI

Drop(拖放)

依赖
脚本文件:CalvinDraggable.js
脚本文件:CalvinDroppable.js
参数
  • options
    Object
    可选

    多选形式的可配置参数。

以下是option参数的说明

参数 数据类型 默认值 说明
onDragEnter Function null

刚进入拖放对象时候触发的方法
onDragOver Function null

在拖放对象上触发的方法
onDragLeave Function null

离开拖放对象时候触发的方法
onDrop Function null

拖拉的对象在拖放的容器内松开鼠标时候触发
accept String null

接收拖放的的元素的选择器用逗号隔开,比如#div1,#div2

实例1

Div1和Div2是拖拉的对象,Div3是拖放的容器只接受Div1

<script type="text/javascript">
            $("#Div1").CalvinDraggable();
            $("#Div2").CalvinDraggable();
            $("#Div3").CalvinDroppable({
                accept: "#Div1",
                onDragEnter: function (e, o) {
                    console.log(o.id + "进入");
                },
                onDragLeave: function (e, o) {
                    console.log(o.id + "离开");
                }
            });
 </script>
