Accrordion(手风琴菜单)

依赖
脚本文件:jquery.js
参数
  • options
    Object
    可选

    多选形式的可配置参数。

以下是option参数的说明

参数 数据类型 默认值 说明
fitParent Boolean false

是否强制停靠父容器,如果为false表示保持原有高度不停靠父容器
initOpenIndex Number 0

初始化打开的面板索引

html结构

  • div class="Accordion"
    必须

    整个组件的外围框架

  • div class="AccordionPanel"
    必须

    面板的容器

  • div class="AccordionPanelTab"
    必须

    面板的标题

  • div class="AccordionPanelContent"
    必须

    面板的内容

实例

基本用法

<script type="text/javascript">
 $("#sampleAccordion").CalvinAccordion({ fitParent: true, initOpenIndex: 1 });
 </script>
